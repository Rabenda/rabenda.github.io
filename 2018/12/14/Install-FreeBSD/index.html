开宗明义

Unix系统就不要搞Xorg这种幺蛾子了

硬件配置

  • ThinkPad t440s
  • ThinkPad t450s（魔改换主板）
  • ThinkPad x1 carbon 6th
  • Hyper-V Virtual Machine 9.0

为什么选用tp (因为信仰) 其实是因为FreeBSD很挑剔硬件 俺不会攒机 直接买TP省心
原来是t440s 后来看到ddr3内存有单条16g 奈何4th的intel不支持单条16g所以只能换主板和内存条了 (x)
后来买了x1c 6th 之后就发生了闲鱼售卖
在之后类Unix系统都安装到Hyper-V里面 省的折腾Xorg

首先是选择安装镜像的问题 FreeBSD官方提供的ISO并不是混合启动盘 所以需要U盘安装的需要下载img后缀镜像
